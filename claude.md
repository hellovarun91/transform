# Varun's Transformation App

A mobile-friendly app combining weekly meal planning and workout tracking.

## Features

### Meals View
- Weekly meal plan (breakfast, lunch, dinner)
- Family members: Varun, Wife, Son (5yr - NO EGGS), Daughter
- Allergy badges: Egg-Free (green), Has Egg (red), Family Meal (blue)
- Sunday prep section

### Workout View
- **Monday/Wednesday/Friday**: Gym days
  - Day A: Push + Core (Mon)
  - Day B: Pull + Core (Wed)
  - Day C: Legs + Full Body (Fri)
- **Tuesday/Thursday/Saturday**: Badminton
- **Sunday**: Rest day
- Morning schedule (6 AM start)
- Daily vitamins reminder

## Goals
- Weight: 90kg -> 80-85kg
- Gym 3x + Badminton 3x/week
- Quit smoking
- High protein diet

## Tech
- Single HTML file with inline CSS/JS
- No dependencies
- Dark mode, mobile-optimized

## Run
```bash
./server.sh
```
